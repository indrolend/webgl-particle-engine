Seamless transitions between solid images and particles

What is Hybrid Particle Transition?

  • Disintegration: Image dissolves into particles with velocity and turbulence
  • Explosion: Particles scatter outward in random directions
  • Recombination: Particles reassemble into target image with vacuum-like attraction
  • Reintegration: Reverse disintegration - particles return to original positions
  • Color Interpolation: Smooth color blending during transitions
  • Unified Rendering: Synchronized image opacity and particle dynamics
Status: Ready

Code Example

// Initialize the HybridPageTransitionAPI
const api = new HybridPageTransitionAPI({
    particleCount: 2000,
    explosionDuration: 800,
    recombinationDuration: 2000,
    autoOptimize: true
});
await api.initialize();

// Start hybrid transition between two images
await api.transitionImages(sourceImage, targetImage, {
    staticDisplayDuration: 500,    // Show solid image
    disintegrationDuration: 1000,  // Dissolve into particles
    explosionIntensity: 150,        // Particle scatter strength
    explosionTime: 800,             // Explosion duration
    recombinationDuration: 2000,    // Reassembly duration
    blendDuration: 1500,            // Blend to final image
    finalStaticDuration: 500        // Show final image
});
